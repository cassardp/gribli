# Gribli

**Match-3 Puzzle Game**

A fun and challenging match-3 puzzle game. Match symbols, chain combos, and climb the leaderboard.

## Features

- Classic match-3 gameplay
- Combo chains and time bonuses
- Global leaderboard
- Simple, polished interface

## Available On

- iOS 17.6+
